# ⚡ Hikareteku RPG 3E — Wiki Interativa & Ficha

> Sistema de RPG de mesa inspirado no universo Dragon Ball, com mecânicas originais de Ki, transformações e progressão de personagem.

## 🌐 Demo

Abra `index.html` diretamente no navegador — **sem dependências, sem build, sem servidor**.

## 📋 Funcionalidades

| Aba | Conteúdo |
|-----|----------|
| ⚙️ **Core Engine** | Regras base, atributos, sistema de dados d10, tipos de Ki e guia de forja de técnicas |
| 🧬 **Raças** | Saiyajin, Namekuseijin, Humano, Androide, Glind — bônus e mecânicas exclusivas |
| 🥋 **Técnicas** | Catálogo completo por mestre (Kame, Kami, Shen, Kuririn, Auto-Didatismo…) com filtros |
| 📈 **Progressão** | Sistema de níveis, Zenkai Boost e evolução de personagem |
| 🔥 **Transformações** | Árvores de transformação por raça (Saiyajin: Oozaru + SSJ1–6; demais: 6 formas cada) |
| 📄 **Minha Ficha** | Criador de ficha interativo com exportação para PDF |

## 🎮 Como jogar

1. Abra `index.html` no navegador
2. Navegue pelas abas para consultar as regras
3. Use a aba **Minha Ficha** para criar seu personagem
4. Use a **barra de busca** (🔍) para encontrar qualquer regra instantaneamente

## 🛠️ Stack

- **HTML5** + **CSS3** (Vanilla — sem frameworks)
- **JavaScript** (Vanilla — sem dependências)
- Fontes: Google Fonts (`Inter`, `Noto Serif JP`, `Fira Code`)
- Tema: Scouter HUD — amber/CRT com dark mode
